Features

Accessible & User-Friendly

Designed to be highly accessible and user-friendly by utilizing APIs for seamless integration with other systems and featuring a intuitive interface that allows for efficient navigation and clear presentation of information to users.

Optimized System

Robust and user-friendly ordering that streamlines the process by providing efficient navigation, clear display of information and options, and an optimized workflow for both customers and servers to place and manage orders.

Responsive

Highly responsive solution that can adapt to any device, by allowing seamless usage across different screen sizes and orientations.

Dynamic

Actively communicates with the database to update inventory and record sales information, resulting in real-time monitoring and management of stock and financial data.

About the Project

In this project I was part of a team that designed and built a custom point of sale (POS) system for Smoothie King. I served as a full-stack developer, working on both the customer and server interfaces as well as back-end logic. My design and programming skills helped bring the project to life. I currently redoing this project on my own and revamping the overall system and front-end interfaces.
